# StaySphere 🚀 🏨

**StaySphere** is a feature-rich, full-stack web application inspired by modern travel platforms like Airbnb. Built with the **MEN stack** (MongoDB, Express, Node.js), it provides a seamless experience for browsing, managing, and reviewing unique lodging listings worldwide.

---

## 🚀 Technologies & Packages Used

### 🛠 Backend & Core
- **Node.js**: Server-side runtime environment.
- **Express.js**: Web framework for building the API and routing.
- **MongoDB & Mongoose**: NoSQL database and object modeling.
- **EJS (Embedded JavaScript)**: For dynamic frontend rendering.

### 🔐 Authentication & Security
- **Passport.js**: For secure user authentication (Local, Google, and Facebook strategies).
- **Joi**: For robust schema description and data validation.
- **Dotenv**: Managing environment variables securely.

### 🌐 API Integrations
- **Mapbox**: Interactive maps for real-time location visualization.
- **Cloudinary**: Cloud-based storage for high-quality listing images.

### 📦 Middleware & Utilities
- **Multer**: Handling multipart/form-data for file uploads.
- **Connect-Flash**: Providing feedback messages to users.
- **Cookie-Parser & Express-Session**: Handling user sessions and cookies.

---

## ✨ Key Features
✅ **User Authentication**: Secure Login, Signup, and Logout functionality.  
✅ **Listing Management**: Full CRUD operations (Add, Edit, Update, Delete listings).  
✅ **Interactive Maps**: Visualize listing locations using Mapbox integration.  
✅ **Review System**: Users can post and delete reviews with star ratings.  
✅ **Social Login**: Seamless authentication via Google and Facebook.  
✅ **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.  
✅ **Image Hosting**: Professional asset management using Cloudinary.

---

## 🕹 How to Use
1. **Explore**: Browse various property listings on the home page.
2. **Search**: Find your next destination using the integrated search bar.
3. **Host**: Sign up/Login to list your own property on StaySphere.
4. **Interact**: View location details on the map and read reviews from other travelers.
